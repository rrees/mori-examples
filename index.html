Mori

Immutable collections for Javascript

What?

Mori is a library that brings Clojure's immutable data structures to Javascript as well as the better part of the Sequence library. It also adds some interop features to allow you to move between mutable (native>) and immutable data structures.

Why?

Mori doesn't require the use of clone().

Memory-reference swaps have become more useful with things like virtual DOMs and other code that detects incremental changes.

How?

When?

One of the most important things about Mori is that we can use it right now in any project that uses Javascript.
